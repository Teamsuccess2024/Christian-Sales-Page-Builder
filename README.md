## What Each File Does
**SKILL.md (master orchestrator)**
The entry point. Tells Claude when to trigger this skill (any time the user asks for Christian sales copy, ebook bundles, faith-based funnels, ministry product pages). Lists the 10 methods, points to references, explains the workflow: intake → choose methods → draft → review against conversion killers.

### references/ — The Knowledge Base

**philosophy.md** — The master frame. Kingdom-first selling: serving the reader is the sale. Avoid manipulation, fake urgency, prosperity-gospel manipulation. Why honesty converts higher with Christians than with secular audiences.

**voice-guide.md** — Christian copy voice: warm, urgent, plain vocabulary, second-person. When to use "brother/sister" language. How to handle different denominations. Words to avoid (overly churchy jargon vs. seeker-friendly). Tone calibration for hurting readers.

**awareness-stages.md** — Eugene Schwartz applied: most Christian buyers in pain are problem-aware but solution-skeptical. How to write for each of the 5 stages without alienating the others.
method-01 through method-10 — One file per method. Each contains: when to use it, the structure, 3-5 examples written in Christian voice, common mistakes, and how it combines with other methods on the same page.

**headline-formulas.md** — 20+ headline patterns proven to work for Christian offers. Curiosity, specificity, scripture-anchored, identity-based. Examples for each.

**scripture-integration.md** — How to use NIV verses persuasively without sounding preachy. Where to place verses on the page. When scripture is the hook vs. the proof vs. the close.

**sales-page-anatomy.md** — The full section-by-section structure of a high-converting Christian sales page. Hero section, story, problem, solution intro, offer reveal, value stack, social proof, FAQ, guarantee, final CTA. Specifies which methods power which sections.

**pricing-positioning.md** — How to frame $27, $47, $97, $297, $497 differently for Christian audiences. Stewardship language. Tithe-equivalency framing. When to offer payment plans.

**conversion-killers.md** — Specific things that tank conversions with Christian buyers: fake countdown timers, prosperity language, "secrets," excessive scarcity, manipulative urgency, theological landmines, denominational triggers.

### templates/ — The Workflows

**intake-questionnaire.md** — Questions Claude asks before writing: What's the offer? Price point? Specific pain addressed? Target denomination range? Author's testimony? What scriptures anchor it? What's the transformation? What's the upsell path?

**full-sales-page-skeleton.md** — A complete sales page template with [BRACKETS] for variables, showing exactly where each of the 10 methods plugs in. Section by section.

**upsell-page-skeleton.md** — Specifically for library/membership upsells like your $297 — different psychology than primary offers (frame shift from solving today's pain to lifetime resourcing).

**headline-generator.md** — Step-by-step process to generate 20 headlines using the formulas, then narrow to top 3.

**voice-calibration.md** — How to match an individual author's voice, theology, and denomination so the copy sounds like them, not generic Christian marketing.

### examples/ — The Worked Cases

**purpose-in-pain-bundle.md** — Your $97 bundle, fully written end-to-end. Real example showing all 10 methods in action. Acts as both a deliverable and a teaching example.

**library-upsell-297.md** — Your $297 full-library upsell page, fully written. Shows the upsell frame-shift in practice.

**single-ebook-27.md** — A lower-ticket single-book sales page example.
coaching-program-497.md — A higher-ticket coaching example showing how the methods scale up.

## How Claude Would Use This Skill

1. Trigger: User asks for Christian sales copy, ebook funnel, ministry product page, faith-based offer.
2. Read SKILL.md first, then load relevant references based on the offer type.
3. Run intake questionnaire to gather offer specifics.
4. Select which of the 10 methods fit this specific offer (not every page needs all 10).
5. Draft using the skeleton template, pulling section structures from method files.
6. Self-check against conversion-killers.md before delivering.
7. Output the finished sales page with notes on which methods were used where.
